# 监督学习
监督学习分为“分类问题”和“回归问题”  
**分类：** 把数据划分成不同的类别  
**回归：** 建立数据间的连续关系  

## 线性回归
1. 最小二乘法  
2. 梯度下降  
3. L1 L2正则化  
4. 数据集扩展  
5. Logistic回归（对数几率回归）  

## 感知器算法
BP（Back Propagation）神经网络
